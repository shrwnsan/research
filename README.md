# Research: Agentic Augmentation

A Jekyll-powered research hub for technical analysis and insights powered by AI tools.

## About

This site contains research papers, technical analysis, and explorations conducted with the assistance of AI agents and tools. The goal is to document findings, share insights, and contribute to the growing body of knowledge about agentic augmentation in development and research workflows.

## Features

- **Jekyll with So Simple Theme**: Clean, minimal design with dark mode support
- **Roboto Slab Typography**: Professional typography matching Stack Overflow survey styling
- **GitHub Pages Ready**: Optimized for deployment on GitHub Pages
- **Mermaid Diagrams**: Support for complex diagrams and visualizations
- **Mobile Responsive**: Works seamlessly across all devices

## Adding New Research

1. Create a new markdown file in `_posts/` following the format: `YYYY-MM-DD-title.md`
2. Add YAML front matter:
   ```yaml
   ---
   title: "Research Title"
   date: YYYY-MM-DD
   categories: [Category1, Category2]
   tags: [Tag1, Tag2, Tag3]
   excerpt: "Brief description of the research"
   ---
   ```
3. Write your research content in Markdown
4. Include Mermaid diagrams using triple backticks with `mermaid` language

## Local Development

This site is optimized for GitHub Pages deployment. For local development, you can use:

```bash
# Using Docker (recommended)
docker-compose up

# Or with Ruby Jekyll (if available locally)
bundle install
bundle exec jekyll serve
```
